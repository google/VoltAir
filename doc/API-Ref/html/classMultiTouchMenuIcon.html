Detailed Description

Icon for use in handling multi-touch events.

The hit area of the icon is defined by width and height properties, but the look and offset of the icon graphic can be changed through the following properties: iconWidth, iconHeight, iconVerticalOffset, and iconHorizontalOffset.

Note
Because MenuIcons use MouseAreas which are fundamentally incompatible with MultiPointTouchAreas, this icon was necessary.
